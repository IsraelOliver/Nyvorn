# v0.4.0 - Mining Progression

Objetivo da versao: transformar a base atual em um loop jogavel simples de minerar, coletar, melhorar ferramenta e explorar um pouco mais fundo.

## Escopo principal

- Blocos com resistencia diferente.
- Ferramentas com tier e poder de mineracao.
- Drops consistentes por tipo de bloco.
- Primeira receita de crafting.
- Primeira melhoria de ferramenta.
- Feedback visual claro ao quebrar blocos.

## Etapa 1 - Dados de blocos

- Criar uma definicao central para blocos mineraveis.
- Cada bloco deve ter:
  - tipo do tile;
  - item dropado;
  - vida/resistencia;
  - tier minimo recomendado;
  - tempo base de quebra.
- Comecar com dirt, grass, stone e sand.

## Etapa 2 - Mineracao com progresso

- Trocar a quebra instantanea por progresso acumulado.
- Mostrar feedback enquanto o bloco esta sendo quebrado.
- Resetar progresso se o player mirar em outro bloco por tempo demais.
- Manter a mao com alcance curto e a picareta com alcance maior.

## Etapa 3 - Tier de ferramentas

- Usar o `PowerTier` da arma/ferramenta para afetar mineracao.
- Picareta tier 1 deve quebrar blocos basicos.
- Blocos acima do tier da ferramenta devem quebrar muito devagar ou nao quebrar.
- Preparar o sistema para tier 2+ sem precisar reescrever a mecanica.

## Etapa 4 - Drops e coleta

- Centralizar o drop por bloco.
- Garantir que itens coletados priorizam hotbar e depois inventario.
- Ajustar quantidades de drop quando necessario.
- Evitar duplicacao de logica entre quebrar bloco e spawnar item no mundo.

## Etapa 5 - Crafting minimo

- Criar uma UI simples para crafting inicial.
- Adicionar pelo menos uma receita de progressao.
- Exemplo inicial:
  - blocos coletados -> recurso refinado ou ferramenta melhorada.
- Salvar o resultado no inventario/hotbar usando as regras atuais.

## Etapa 6 - Polimento

- Adicionar efeito visual simples ao bater no bloco.
- Adicionar som ou placeholder de som para mineracao.
- Ajustar sensacao da picareta: alcance, tempo, dano baixo e knockback.
- Revisar se a UI da hotbar e inventario continuam legiveis.

## Fora do escopo por enquanto

- Sistema completo de crafting avancado.
- Muitas armas novas.
- Biomas grandes novos.
- Chefes ou inimigos complexos.
- Arvores, agricultura ou NPCs.

## Criterio para fechar v0.4.0

- O player consegue minerar blocos com progresso perceptivel.
- A picareta funciona melhor que a mao.
- Blocos dropam itens corretamente.
- Existe pelo menos uma acao de progressao usando crafting ou melhoria.
- O jogo compila e a branch pode ser mergeada para `main`.
